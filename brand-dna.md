# 🎨 品牌色彩系统（Brand Color System）

## 品牌主色（80%视觉权重）

| 色名  | 色值        | 对应   | 用途              |
| --- | --------- | ---- | --------------- |
| 墨青蓝 | `#2F4156` | 品牌主色 | 标题、大面积重点区域、主视觉  |
| 云灰蓝 | `#708A9A` | 辅助色  | 卡片背景、分割模块、图表辅助色 |

# 🔤 字体基因

## 核心原则

* 中文优先
* 标题用衬线
* 正文用无衬线
* 留白比装饰重要
* 少即是多

---

## 推荐字体池

| 场景     | 推荐                   | 备注       |
| ------ | -------------------- | -------- |
| 中文主标题  | Huiwen Mincho（汇文明朝体） | 第一优先     |
| 中文标题备选 | Noto Serif SC 900    | fallback |
| 中文正文   | Noto Sans SC         | 默认正文     |
| 英文标题   | Cormorant Garamond Italic     | 艺术感      |
| 英文注释   | Caveat               | 手写感      |
| 技术内容   | Fira Code            | 代码场景     |

---

# 🎯 品牌自检问题

每次设计完成后必须检查：

### 01

这个页面截图发到设计社区，

会不会有人说：

"又是一个AI生成的模板？"

如果会。

重做。

---

### 02

遮住Logo以后，

还能不能认出来这是珠珠子的内容？

如果不能。

重做。

---

### 03

这个设计除了好看之外，

有没有表达一个明确观点？

如果没有。

重做。

---

### 04

这个页面像作品，

还是像模板？

优先成为作品。

---

### 05

五年以后再看，

是否仍然成立？

如果明显追逐潮流。

重做。

# 📐 通用间距原则

`clamp(96px, 12vh, 180px)`

---

内容块之间：

`clamp(48px, 6vw, 120px)`

---

模块内部：

`clamp(32px, 3vw, 56px)`

---

元素间距：

`clamp(24px, 3vw, 48px)`

宁可松一点。

不要挤。

留白是内容的一部分。

# 🌿 最终审美标准

不追求惊艳。

不追求流量。

不追求复杂。

安静、克制、真实、有思考。

让设计像一本被反复翻阅的书，
