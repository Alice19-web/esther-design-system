# 质量检查清单

> 页面完成后逐条检查。
>
> P0必须全部通过，否则返回修改。

---

# P0（必须全过）

任何一项不通过：

```text
直接返工
```

---

* [ ] 色彩体系完全继承 brand-dna.md
* [ ] 未新增任何品牌色
* [ ] 未使用禁忌元素（蓝紫渐变 / Neon / Cyan / 科技感发光效果 / Bounce动画）
* [ ] 页面背景使用宣纸白体系（--cream）而非纯白纯黑
* [ ] 字体系统符合品牌规范（汇文明朝体 / Noto Serif SC / Noto Sans SC）
* [ ] 标题使用衬线字体，正文使用无衬线字体
* [ ] 响应式完整（至少包含 900px 与 600px 两个断点）
* [ ] 使用 clamp() 实现流体字号
* [ ] 页面具有明确的信息层级
* [ ] 不依赖浏览器默认样式
* [ ] 所有组件来自 design-system 或 components.md
* [ ] 页面整体符合「明月Moon」品牌气质
* [ ] 截图后看起来像专业设计作品，而非AI模板

---

# P1（建议通过）

尽量满足。

---

* [ ] 至少存在一个视觉焦点
* [ ] 至少存在一种非对称布局
* [ ] 大小字号形成明显层级差异
* [ ] 使用 Scroll Reveal 动效
* [ ] Reveal 动效包含合理 Stagger 延迟
* [ ] 存在大装饰数字或大装饰文字
* [ ] 使用禅金色进行局部强调
* [ ] 至少一个模块具备出版物设计感
* [ ] 页面具有明确阅读节奏
* [ ] 页面存在适量留白

---

# P2（加分项）

优秀作品建议具备。

---

* [ ] 图片突破容器边界（出血设计）
* [ ] 存在全宽品牌色模块打破节奏
* [ ] 使用条纹肌理、光晕或装饰框等细节元素
* [ ] 动效遵循 prefers-reduced-motion
* [ ] Pull Quote 使用得当
* [ ] 图片与文字形成杂志式编排
* [ ] 存在具有记忆点的视觉细节
* [ ] 页面具备分享传播价值

---

# 品牌一致性检查

最终检查：

* [ ] 像明月Moon，而不是互联网产品官网
* [ ] 像设计出版物，而不是运营长图
* [ ] 像艺术馆导览，而不是AI生成模板
* [ ] 像长期品牌资产，而不是一次性活动页

---

# 最终目标

完成后的页面应具备：

```text
温暖

克制

理性

高级

出版物感

艺术馆感

东方现代主义
```

而非：

```text
科技感

赛博朋克

运营风

AI模板风

互联网大厂风
```
